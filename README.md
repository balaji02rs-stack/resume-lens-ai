# ResumeLens AI

AI-powered Resume Analysis Platform built using **Java, Spring Boot, React, PostgreSQL, JWT Authentication, Docker, and REST APIs**. The application enables users to upload resumes, perform ATS score analysis, compare resumes with job descriptions, and securely manage resume history.

---

## 🚀 Live Demo

**Application:** [https://YOUR-FRONTEND-URL](https://resume-lens-frontend.onrender.com)

**Backend API:** https://resume-lens-backend-bxb8.onrender.com

---

## ✨ Features

- Secure User Registration & Login (JWT Authentication)
- Resume Upload (PDF)
- Resume Parsing using Apache PDFBox
- ATS Score Analysis
- Job Description Matching
- Resume History Management
- PostgreSQL Database Integration
- RESTful APIs
- Responsive React Frontend
- Cloud Deployment on Render

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/resume-lens-ai.git
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
