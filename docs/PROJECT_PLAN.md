# ResumeLens AI

## Vision

ResumeLens AI is an AI-powered career platform that helps job seekers improve their resumes, understand ATS compatibility, match resumes against job descriptions, and receive practical suggestions without hiding important information behind subscriptions.

---

## Goals

- Simple and clean interface
- Free resume analysis
- ATS score explanation
- Job description matching
- Skill gap analysis
- AI-powered suggestions
- Fast performance
- Mobile-friendly
- Privacy-focused

---

## Version 1 Features

- Upload Resume (PDF/DOCX)
- Paste Job Description
- Resume Parsing
- ATS Score
- Keyword Matching
- Missing Skills
- Resume Feedback
- Download Report
